📊 Super Store Sales Dashboard
📌 Overview

This project is an interactive business intelligence dashboard built using Microsoft Power BI Desktop.
It provides detailed insights into sales, profit, quantity, and customer behavior across different regions and categories.

🚀 Key Features
💰 Total Sales: 621K
📦 Total Quantity: 9119
📈 Total Profit: 70K
📊 Average Sales Value: 53
📍 Dashboard Insights
🔹 Sales Analysis
Sales distribution by Segment (Consumer, Corporate, Home Office)
Sales breakdown by Category (Office Supplies, Technology, Furniture)
Top performing Sub-Categories (Phones, Binders, Chairs)
🔹 Time-Based Analysis
📅 Monthly Sales Trends (Year-wise comparison)
📉 Monthly Profit Trends
🔹 Regional Insights
🌍 Sales & Profit visualization by State (Map View)
Region filter (Central, East, South, West)
🔹 Customer Behavior
💳 Payment Mode Analysis (COD, Cards, etc.)
🚚 Shipping Mode Analysis (Standard, First Class, Same Day)
